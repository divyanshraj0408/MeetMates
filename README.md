# RoomBall

RoomBall is a web application designed exclusively for the students of Dr. Akhilesh Das Gupta Institute of Professional Studies. It allows students to connect with each other through random video calls, similar to Omegle. The platform aims to foster connections among students, helping them meet new people and share interests in a secure environment.

## Features

- **Random Video Matching**: Students are matched with each other randomly for video calls.
- **Secure Access**: Only students from Dr. Akhilesh Das Gupta Institute of Professional Studies can sign up and use the platform.
- **Real-time Communication**: Utilizes WebRTC for high-quality video and audio communication.

## Future Scope

- **Friendship System**: Allow users to add each other as friends after a match.
- **Chat Functionality**: Implement text chat for continued conversation even after the video call ends.
- **Preference Matching**: Users can set preferences based on interests, allowing them to match with like-minded peers.

## Tech Stack

- **Frontend**: React
- **Backend**: Node.js, Express
- **Real-time Communication**: Socket.io, WebRTC
- **Database**: TBD (potentially MongoDB or Firebase)
